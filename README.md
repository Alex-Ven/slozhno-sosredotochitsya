# Hard to Focus — Why We Can't Concentrate (Educational Project)

**A beautifully designed, fully responsive article** about the science of focus, distractions, and practical tips to improve concentration.

**This is my educational project** — built to master **advanced CSS theming, BEM, and responsive design**.

Live Demo: [slozhno-sosredotochitsya.vercel.app](https://slozhno-sosredotochitsya.vercel.app)

---

## Technologies

| Category           | Technologies |
|--------------------|--------------|
| **Language**       | HTML5, CSS3, Vanilla JavaScript |
| **Theming**        | 3 themes: **Light**, **Dark (Neon)**, **Auto** |
| **Styling**        | CSS Variables, BEM, modular styles |
| **Responsive**     | Mobile-first, fluid layout, media queries |
| **UX**             | Theme persistence, smooth transitions |

---

## Features

- **3 Visual Themes**:
  - **Light** (Day)
  - **Dark / Neon** (high contrast)
  - **Auto** (system preference)
- **Responsive Design** — works perfectly on **mobile, tablet, desktop**
- **Theme Switcher** with `localStorage` save
- **Rich Typography** with custom fonts
- **Scientific References** with external links
- **Image Gallery** with lazy loading
- **Decorative Zones** with CSS pseudo-elements

---

## Theme System

```css
:root { --accent-color: #ff0070; }
.theme_dark { --accent-color: #00f7ff; }
```

```js
localStorage.setItem('theme', 'dark');
```

- Themes in separate CSS files
- Smooth transitions
- `disabled` state for active button
